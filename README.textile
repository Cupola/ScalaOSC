h3. requirements / installation

ScalaOSC currently compiles against the Scala 2.8.0.RC1. The repository contains a project file for Intellij IDEA 9 CE, and project files for sbt. Standard targets for sbt:

* @clean@
* @compile@
* @doc@
* @package@
* @run@ __arguments...__
